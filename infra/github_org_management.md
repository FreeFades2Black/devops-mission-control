# GitHub Organization & Settings

## 1. Leaving an Organization
1. Profile Icon -> **Settings**
2. Sidebar -> **Organizations**
3. Select Org -> **Leave**

## 2. Managing Notifications
If the organization is sending you too many emails, you can stay in the org but turn off the noise:
* **Settings** -> **Notifications** -> **Custom Categories**

## 3. Terminal Navigation
| Shortcut | Action |
| :--- | :--- |
| `git remote -v` | Verify your current project isn't accidentally pushing to an Org repo. |
| `gh org list` | (If you have GitHub CLI installed) List your organizations in the terminal. |

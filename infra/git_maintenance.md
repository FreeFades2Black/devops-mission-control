# Git Maintenance & Troubleshooting

## 1. Identity Verification
| Command | Action |
| :--- | :--- |
| `git config --list` | See every setting currently applied to Git. |
| `git config user.email` | Quickly check your active email. |

## 2. Managing Hidden Files
* **Swap Files:** `.swp` files are created by editors like `vi` or `nano`. Delete them if the editor isn't open.
* **Dot Files:** Files starting with `.` (like `.gitignore`) are usually configuration files.

## 3. Terminal Shortcuts
| Shortcut | Action |
| :--- | :--- |
| `Ctrl + L` | Clear the screen to get a fresh start. |
| `up arrow` | Cycle through your previous commands to find the `git config` you typed. |

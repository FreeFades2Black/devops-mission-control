# GitHub Fast Navigation

## 1. Get Me Out of the Dashboard!
| Goal | Action |
| :--- | :--- |
| **View My Progress** | Click Profile Icon -> "Your profile" |
| **Go to My Project** | Click "devops-mission-control" under "Top Repositories" on the left. |
| **Direct Link** | Bookmark: github.com/FreeFades2Black |

## 2. Keyboard Shortcuts (Save Time)
| Shortcut | Action |
| :--- | :--- |
| `g` then `p` | Jump to your Profile page instantly. |
| `g` then `r` | Jump to your Repositories list. |
| `/` | Focus the search bar to find a specific file. |

## 3. GitHub Habit
* **Check the Graph:** Always check your profile *after* a merge to main. If you don't see green, verify that "Private contributions" is turned on in your profile's contribution settings.
